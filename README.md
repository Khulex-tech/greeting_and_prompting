# 🛡️ CyberShield SA – Cybersecurity Awareness Chatbot

## 📌 Overview
CyberShield SA is a **C# console-based chatbot** designed to teach users about **basic cybersecurity practices**.  
The chatbot answers questions about online safety and provides helpful tips to help users stay secure on the internet.

---

## ⚙️ Features

- 💬 Interactive chatbot conversation
- 🔐 Information about cybersecurity topics
- 🎲 Random cybersecurity tips
- 📜 Conversation history tracking
- 🔊 Welcome audio when the program starts
- 🖼️ ASCII logo displayed in the console

---

## 🧠 Topics Covered

Users can ask about:

- Password Safety 🔑  
- Phishing Scams 🎣  
- Malware & Viruses 🦠  
- VPN Security 🌐  
- Two-Factor Authentication (2FA) 📲  
- Social Engineering 🧑‍💻  
- Public WiFi Safety 📡  
- Privacy Protection 🔒  

---

## ⌨️ Commands

| Command | Description |
|--------|-------------|
| `tip` | Get a random cybersecurity tip |
| `history` | Show the conversation history |
| `exit / quit / bye` | Close the chatbot |

---


---

## ▶️ How to Run

1. Open the project in **Visual Studio**
2. Make sure the **assets folder** contains the audio and logo
3. Build and run the project
4. Enter your name and start chatting with the bot 💬
